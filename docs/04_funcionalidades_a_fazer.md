# Funcionalidades a Fazer

Esta seção lista as funcionalidades planejadas para o futuro, que ainda não foram iniciadas.

---

## 1. Módulo de Vendas - Pedidos e Orçamentos

**Prioridade:** Alta

**Descrição:** Implementar um fluxo completo para criação de pedidos de venda, orçamentos, e gestão do ciclo de vida da venda, incluindo emissão de notas fiscais (futuro).

**Impacto:** Core para o negócio.

**Requisitos Preliminares:**
*   Módulo de Produtos completo.
*   Integração com módulo Financeiro.

---

## 2. Relatórios e Dashboards Dinâmicos

**Prioridade:** Média

**Descrição:** Desenvolver uma interface flexível para criação e visualização de relatórios personalizados e dashboards interativos, utilizando os dados existentes no sistema.

**Tecnologias Potenciais:** `Recharts`, `Chart.js`, etc.

---

## 3. Integração com ERP Externo

**Prioridade:** Baixa (mas estratégica)

**Descrição:** Estudar e planejar a integração com sistemas ERP externos para sincronização de dados (produtos, clientes, vendas).
